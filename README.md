# Eucaliptus AR — Landing (React + Tailwind)

Landing page moderno, responsive y optimizado para SEO creado con React + Vite + TailwindCSS.

Cómo ejecutar:

1. Instalar dependencias

   npm install

2. Iniciar el servidor de desarrollo

   npm run dev

Notas:
- Diseño pensado para claridad, velocidad y storytelling visual.
- Colores y tono basados en el sitio actual y documentos institucionales.
- Formulario de contacto es estático y puede conectarse a un backend o servicio de formularios.
